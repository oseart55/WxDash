  • Basics

  • How do I move the Map?

    Select any location within the Map, click, and drag to move the map. Utilize either the mouse scroll wheel if you have one, or the buttons on the map to Zoom In or Out. You may also, while holding the SHIFT button on your keyboard, click and drag the mouse to create a Bounding Box that the map will then zoom to.

  • How do I change the Map Basemaps/Layers?

    Utilize the Layer Control located at the Top Right of the Map. Hover over the Layer Control and select any of the Basemaps or Layers to add or remove them from the Map. Further information about the Vector Layers can be found by selecting the "More Info" tag on each Layer.

  • How often is the data updated?

    Each Vector Layer will update depeding on the sources update time table, generally every 10-15 minutes. Please reference information for the layer that you are interested in to learn more.

  • Dashboard Elements

  • How do I search for Features?

    Once you have selected a Vector Layer from the Layer Control, all Features will be loaded into the Dashboard Collection List. You may then use the Filter Text Box and entery a Term to search for. While typing, Features matching the term will be displayed in a dropdown to select.

  • When do the Collection List and Indicator update?

    The Collection List and Indicators will be updated as you interact with the Data on the layer that is selected. Moving the Map or Filtering will cause these elements to update.

  • Toolbox Tools

  • How do I use the "Get Current Weather Watches/Warnings" tool?

    Once you have clicked on the Toolbox icon, select the radio button for the "Get Current Weather Watches/Warnings" tool. Then, select a location on the map and you will get current Weather Watches/Warnings for that location.
